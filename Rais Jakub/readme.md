# Hledání min (Minesweeper)

Tento projekt vznikl jako samostatná maturitní práce z programování (4. ročník).  
Hra je vytvořena v jazyce **Python** pomocí knihovny **Tkinter** a využívá **SQLite databázi** pro ukládání výsledků.

## Popis hry
Cílem hry je odkrýt všechna pole, která neobsahují minu.  
Pokud hráč klikne na minu, hra končí prohrou.  
Pravým tlačítkem myši lze označit pole vlaječkou, které si hráč myslí, že skrývá minu.

Každá obtížnost má jiný počet min a rozměr hrací plochy.  
Hra se ovládá pouze myší.
